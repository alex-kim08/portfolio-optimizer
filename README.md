# portfolio-optimizer
This project implements the Modern Portfolio Theory to help optimize portfolios.   
It uses historical stock data from the yfinance api to calculate expected returns and risk, finding the most efficient use of capital.

Features:  
Automated Data Retrieval  
Monte Carlo Simulation  
The Efficient Frontier  
Sharpe Ratio Optimization
